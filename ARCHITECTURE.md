## Security Model

### Defense in Depth

1. **API Layer**
   - CORS origin enforcement
   - Rate limiting (120 req/min default)
   - JWT authentication with refresh tokens
   - API key rotation with Argon2 hashing

2. **Substrate Layer**
   - Immutable invariant guard (blocks `auto_apply=True` in production)
   - Circuit breaker on constraint violations
   - Rollback history for all autonomous actions
   - Fail-closed on evidence seal mismatch

3. **Data Layer**
   - SQLAlchemy ORM prevents SQL injection
   - Secrets via environment variables (never in code)
   - Audit logs with user attribution
   - Soft delete with temporal validity
