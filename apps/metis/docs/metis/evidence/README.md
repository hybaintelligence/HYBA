## Generation Commands

```bash
# 1. Real data benchmark
PYTHONPATH=apps/metis/src:src python apps/metis/scripts/metis_real_data_benchmark.py \\
  --seeds 11 17 23 29 31 \\
  --output apps/metis/docs/metis/evidence/metis_real_data_benchmark.json \\
  --dashboard-output apps/metis/docs/metis/evidence/metis_dashboard_summary.json

# 2. Quantum performance benchmark
PYTHONPATH=apps/metis/src:src python apps/metis/scripts/metis_quantum_performance_benchmark.py \\
  --output apps/metis/docs/metis/evidence/metis_quantum_performance_benchmark.json

# 3. Report check
PYTHONPATH=apps/metis/src:src python apps/metis/scripts/metis_benchmark_report_check.py \\
  apps/metis/docs/metis/evidence/metis_real_data_benchmark.json \\
  --output apps/metis/docs/metis/evidence/metis_report_check.json

# 4. Metrics export
PYTHONPATH=apps/metis/src:src python apps/metis/scripts/metis_metrics_export.py \\
  apps/metis/docs/metis/evidence/metis_real_data_benchmark.json \\
  --output apps/metis/docs/metis/evidence/metis_metrics_standard_report.json

# 5. Training readiness
PYTHONPATH=apps/metis/src:src python apps/metis/scripts/metis_training_readiness.py \\
  --benchmark-report apps/metis/docs/metis/evidence/metis_real_data_benchmark.json \\
  --quantum-report apps/metis/docs/metis/evidence/metis_quantum_performance_benchmark.json \\
  --metrics-standard-report apps/metis/docs/metis/evidence/metis_metrics_standard_report.json \\
  --output apps/metis/docs/metis/evidence/metis_training_readiness.json
```

## Notes

- All computations are **substrate independent** and **hardware agnostic**
- Quantum comes from pure mathematics (Yang-Mills, dodecahedral, Pulvini)
- No quantum advantage claims are made without evidence-backed gates
- Each artifact is reproducible by re-running the generation command
- Artifacts can be verified against their SHA-256 fingerprints
