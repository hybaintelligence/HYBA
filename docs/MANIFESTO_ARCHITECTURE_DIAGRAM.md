## VISUAL SPECIFICATION FOR DIAGRAM GENERATION

### Color Scheme

- **Mathematical Foundation Layer**: Deep Blue (#003366)
- **Axiom Layers**: Gradient from Blue to Purple
  - Axiom 1: Blue (#0066CC)
  - Axiom 2: Indigo (#3333CC)
  - Axiom 3: Purple (#6600CC)
  - Axiom 4: Magenta (#CC0066)
  - Axiom 5: Red (#CC3300)
- **Evidence Governance**: Gold (#FFCC00)
- **Application Layer**: Green (#00CC66)
- **Substrate Layer**: Gray (#666666)

### Component Shapes

- **Modules**: Rounded rectangles with 3px border
- **Data Flows**: Solid arrows with arrowheads
- **Dependencies**: Dashed lines
- **Cross-Cutting Concerns**: Horizontal bands spanning all layers
- **External Systems**: Cloud shapes

### Icons

- **Mathematical**: Σ (summation symbol)
- **Cryptographic**: 🔒 (lock)
- **Sovereign**: 🏛️ (building/columns)
- **Substrate**: 🔧 (gear)
- **Evidence**: 📋 (clipboard)
- **Validation**: ✓ (checkmark)

### Layout Principles

1. **Vertical Stack**: Axioms stacked from bottom (Axiom 1) to top (Axiom 5)
2. **Horizontal Flow**: Data flows left to right
3. **Cross-Cutting**: Governance and validation span horizontally
4. **Modularity**: Each axiom is a self-contained module
5. **Interconnection**: Clear arrows showing dependencies
