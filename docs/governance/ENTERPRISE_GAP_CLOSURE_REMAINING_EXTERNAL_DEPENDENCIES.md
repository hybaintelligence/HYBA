# Remaining External Dependencies

The repository implementation closes code and operating-control gaps, but the following items require infrastructure or third-party execution:

- External SOC2 audit.
- External penetration test.
- Production SIEM integration.
- Production paging/on-call integration.
- Enterprise SSO provider configuration.
- Public/customer status page provisioning.
- Cloud billing budgets and anomaly alerts.
- Multi-region failover deployment and game-day evidence.
- Production load test against deployed infrastructure.

These must be tracked separately and must not be represented as completed by this branch alone.
