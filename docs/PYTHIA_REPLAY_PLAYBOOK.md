# PYTHIA Replay Reproducibility Playbook

## What this is

The replay stack is a local, deterministic evidence pipeline for mining and
benchmark claims. It helps developers prove that a local claim can be regenerated
from recorded inputs, commands, seeds, pins, and artifacts.

## Common CLI workflows

```bash
# Replay a manifest claim
PYTHONPATH=python_backend python scripts/replay_claim.py replay manifest.json --cwd /path/to/run

# Stress-test determinism
PYTHONPATH=python_backend python scripts/replay_claim.py stress manifest.json --runs 10 --cwd /path/to/run

# Run one falsification route
PYTHONPATH=python_backend python scripts/replay_claim.py falsify manifest.json seed_mutation_negative_control --cwd /path/to/run

# Run all falsification routes
PYTHONPATH=python_backend python scripts/replay_claim.py falsify-all manifest.json --cwd /path/to/run

# Write audit reports
PYTHONPATH=python_backend python scripts/replay_claim.py report manifest.json --cwd /path/to/run \
  --report-json replay-report.json --report-html replay-report.html

# Register a verified manifest and artifacts
PYTHONPATH=python_backend python scripts/replay_claim.py register manifest.json .replay-registry --cwd /path/to/run

# List and reverify registry entries
PYTHONPATH=python_backend python scripts/replay_claim.py list .replay-registry
PYTHONPATH=python_backend python scripts/replay_claim.py reverify .replay-registry claim_id
```

## Generator workflow

Mining or benchmark code should call
`pythia_mining.emit_attested_mining_success_manifest()` after a successful local
run. The helper emits a manifest fragment with boundaries, commands, seeds,
dependency pins, artifact declarations, an expected output digest, and a seed
mutation falsification route.

## Review checklist

- Manifest includes explicit boundaries.
- `reproducibility_attestation` was generated by the builder, not hand-edited.
- Declared artifacts are small enough to store and are regenerated by replay.
- Stress tests pass for the required run count.
- All declared falsification routes diverge as expected.
- External materials do not convert local replay evidence into revenue,
  pool-side acceptance, regulatory, or consciousness claims.

## Install locally

```bash
python -m pip install -e .
pythia-replay replay examples/replay_nonce/manifest.json --cwd examples/replay_nonce
```

## Try bundled examples

```bash
# Deterministic local nonce proof artifact
PYTHONPATH=python_backend python scripts/replay_claim.py replay examples/replay_nonce/manifest.json \
  --cwd examples/replay_nonce

# Deterministic matrix summary artifact
PYTHONPATH=python_backend python scripts/replay_claim.py replay examples/replay_matrix/manifest.json \
  --cwd examples/replay_matrix

# Register and reverify an example
PYTHONPATH=python_backend python scripts/replay_claim.py register examples/replay_nonce/manifest.json \
  .replay-registry --cwd examples/replay_nonce
PYTHONPATH=python_backend python scripts/replay_claim.py reverify .replay-registry example_nonce_replay
```
