# Enterprise GTM and SOC 2 Readiness

## SOC 2 program

- Hire a SOC 2 auditor and define Type I / Type II audit windows.
- Document security controls, access logging, incident response, data retention, and disaster recovery.
- Map controls to evidence owners and automated collection paths.

## Customer success program

1. Contract signature and tenant provisioning.
2. API key issuance and documentation handoff.
3. Technical onboarding call and first workload execution.
4. Monthly check-in, churn-risk review, and quarterly business reviews.

## Partner ecosystem

- System integrators: Accenture, Deloitte, IBM Consulting.
- Marketplaces: AWS Marketplace, Azure Marketplace, and Google Cloud Marketplace.
