This document does **not** claim a metaphysical proof that the physical universe is computable. ## Doctrine

**Nodus Solutus** means the knot is untied only when the claim has been converted into a runnable evidence structure.

**Mundus Computabilis Est** is used here as a bounded engineering thesis:

> The repository's operational world is computable when its claims are represented as code, tests, replayable evidence artifacts, and falsifiable boundaries.

The phrase is not used as an unbounded claim about physics, metaphysics, consciousness, revenue, solvency, law, or the total physical universe.

## Claim escalation path

1. **Text claim** — a phrase appears in docs, README material, release notes, or operator language.
2. **Manifest claim** — the phrase is entered into `docs/evidence/claim_evidence_manifest.json` with a bounded claim, code paths, test paths, doc paths, and commands.
3. **Executable claim** — a local command can reproduce the evidence from a clean checkout.
4. **Gate claim** — `npm run review:evidence:gate` and `npm run prod:local:gate` include or transitively include the proof surface.
5. **Artifact claim** — the local production gate writes a timestamped JSON evidence packet with command results and SHA-256 digest.
6. **External claim** — revenue, pool acceptance, treasury, legal, or production economics remain blocked until external truth data is attached.
