## The 4 Intelligence Functions

### 1. PREDICT
**Method:** Deutsch Counterfactual Reasoning

Uses the Knowledge Substrate to predict outcomes based on past explanations and counterfactual models.

**Example:**
```bash
curl -X POST http://localhost:3001/api/qiaas/query \
  -H "Content-Type: application/json" \
  -d '{
    "query_type": "predict",
    "context": {
      "difficulty": 1000000,
      "thermal_load": 0.5,
      "phi_resonance": 0.618
    }
  }'
```

### 2. EXPLAIN
**Method:** Popperian Conjecture and Criticism

Generates explanations for why strategies succeed or fail, then subjects them to criticism against new data.

**Example:**
```bash
curl -X POST http://localhost:3001/api/qiaas/query \
  -H "Content-Type: application/json" \
  -d '{
    "query_type": "explain",
    "context": {
      "strategy_id": "golden_ratio_substrate"
    }
  }'
```

### 3. OPTIMIZE
**Method:** Constructor Theory Counterfactuals

Generates optimization proposals by reasoning about what would happen under alternative strategies.

**Example:**
```bash
curl -X POST http://localhost:3001/api/qiaas/query \
  -H "Content-Type: application/json" \
  -d '{
    "query_type": "optimize",
    "context": {
      "current_strategy": "baseline",
      "alternative_strategy": "phi_optimized"
    }
  }'
```

### 4. HEAL
**Method:** Salamander Regeneration

Triggers quantum regeneration (blastema formation + redifferentiation) for degraded components.

**Example:**
```bash

  -H "Content-Type: application/json" \
  -d '{
    "query_type": "heal",
    "context": {
      "component_id": "quantum_solver",
      "lane_id": 0
    }
  }'
```
