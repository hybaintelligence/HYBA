## HYBA documents Nodus Solutus, a local-first evidence doctrine for deterministic Bitcoin and frontier-computation research

HYBA Analytics Ltd has documented **Nodus Solutus: Mundus Computabilis Est**, a repository-local evidence doctrine for HYBA_FULLSTACK. The doctrine requires major technical claims to be backed by deterministic source paths, executable tests, replayable local evidence packets, and explicit external-truth boundaries.

The doctrine is designed to support two separate tracks:

1. **Bitcoin operational readiness** — validated by the local Bitcoin deployment gate.
2. **Frontier scientific research** — validated by separate research/elevation gates.

This separation allows production work to proceed without overstating research results, while preserving an auditable path for frontier experiments.

## Operator validation

The Bitcoin deployment readiness path is:

```bash
npm run review:nodus:gate
npm run review:manifest:gate
npm run prod:bitcoin:gate
```

The frontier research path is:

```bash
npm run test:frontier:experiments
npm run test:elevation:suite
npm run prod:research:gate
```

A passing Bitcoin gate is a local release-candidate evidence packet. It is not authorization to enable live share submission. Live share submission requires explicit operator approval and pool-side evidence.
