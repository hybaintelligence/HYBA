## STATISTICAL METHODOLOGY

### Significance Testing

All tests use binomial proportion testing:

```python
z = (p_hat - p₀) / sqrt(p₀(1-p₀)/n)
```

where p_hat = observed success rate, p₀ = null hypothesis rate (typically 0.5), n = number of trials.

Two-tailed p-values computed using Abramowitz-Stegun normal CDF approximation (accurate to 1.5×10⁻⁷).

### Effect Size

Cohen's h for two proportions:
```
h = 2 arcsin(sqrt(p₁)) - 2 arcsin(sqrt(p₂))
```
